# Discord Timestamp Generator

A timezone-aware Discord timestamp generator built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui. Select any date, time, and timezone to preview Discord's timestamp chat syntax and copy ready-to-use snippets.

## Features
- Calendar picker for choosing the event date
- Native time input for precise hours and minutes
- Timezone selector populated from your browser's supported timezones
- Live previews for all Discord timestamp display codes (t, T, d, D, f, F, R)
- Quick copy buttons for syntax and raw UNIX timestamps

## Development
Install dependencies and start the development server:

```bash
npm install
npm run dev
```

The app runs at `http://localhost:3000`.
